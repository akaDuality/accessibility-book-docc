# Практика

Упражнения для освоения VoiceOver — включите его и попробуйте пользоваться телефоном.

## Задания

Выполните следующие упражнения для закрепления:

1. **Включи VoiceOver.** Для выключения можно дать команду Сири, проверь заранее, что она работает.

2. **Настрой шорткат** на быстрый доступ через кнопку включения или на тап по спинке телефона.

3. **Попользуйся стандартными приложениями iOS**, посмотри как они читают кнопки, их названия и типы.

4. **Научись сворачивать приложения**, открывать панель с оповещениями.

5. **Попробуй отключить экран** и ориентироваться только на звук.

6. **Попробуй попользоваться своим приложением**, отметь непонятные места, где нет подписей, где много контролов и где сломался фокус.

7. **Получится ли выполнить основной сценарий** в твоём приложении?

## Что дальше

На этом этапе стоит записывать все непонятные места в вашем приложении. Позже вы поймёте, как их починить.

> Tip: Если нет своего приложения, то возьми любое популярное приложение из App Store. Пример хорошей адаптации можно посмотреть в приложении Додо Пиццы.
