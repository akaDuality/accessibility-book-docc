# Actions

The main VoiceOver gestures for interacting with elements.

## Taps and swipes

Taps and swipes are used to control VoiceOver. You can tap anywhere — the command will go to *the currently focused element*.

## Single tap — read

**Read the element description**. A *single tap* on the screen performs the most important action — **it reads the element description.** It's not used often, since the description is read automatically on focus change.

## Double tap — activate

**Press the button** by *double tapping* anywhere on the screen — the action will be sent to the focused element.

## Closing the app

Try opening an app with a double tap. If you have an iPhone "with a notch", it's closed with the familiar swipe from the bottom, only slightly slower: feel the light vibration, then release your finger and the program will minimize. If you do it "wrong," the app switcher will open.

## Three-finger scroll

**Scroll the screen** by *swiping three fingers* in the desired direction. Try flicking through the iPhone's home screen with the icons.

## Turning off the screen

If you want to immerse yourself in how blind people interact with interfaces, then **turn off the screen** by *triple tapping with three fingers*. You turn it back on the same way.

## Reading all elements

When getting to know a new program, you often need to **read all the elements** on the screen. If you *swipe two fingers up*, VoiceOver will read all the elements from the beginning of the screen, and if you *swipe down* — starting from the current one.

## Other gestures

We've gotten to know a few basic gestures, but VoiceOver has many more. A few popular ones can be seen on the next page, and the full list is [in the guide](https://a11y-guidelines.orange.com/en/mobile/ios/voiceover/).

![VoiceOver gesture reference.](gestures-reference)

> Note: In some cases the vertical swipe also works, but we'll talk about that later — in the chapters "Adjustable element", "Custom actions" and "Rotor".

| Gesture | Action |
|------|----------|
| Four-finger tap at the top | To the first element |
| Four-finger tap at the bottom | To the last element |
| Two-finger zigzag | Leave the screen |
| One-finger swipe down | Read everything from the current point |
| Two-finger swipe up | Read everything from the first |
| Three-finger swipe down | Scroll down |
| Two-finger double tap | Mute VoiceOver |
| Three-finger triple tap | Turn off the screen |
| Two-finger rotation | Rotor |
